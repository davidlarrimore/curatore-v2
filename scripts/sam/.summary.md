## Quick Reference

**Run SAM.gov pull script:**
```bash
# Method 1: Docker (easiest)
docker-compose exec backend python /app/scripts/sam/sam_pull.py

# Method 2: Backend venv
cd backend && source venv/bin/activate && cd ..
python scripts/sam/sam_pull.py

# Method 3: Direct python3
python3 scripts/sam/sam_pull.py
```

**View documentation:**
- Main: `cat scripts/sam/README.md`
- How-to: `cat scripts/sam/HOW_TO_RUN.md`
- Config: `cat .env.review.md`

**Schedule daily runs:**
```bash
crontab -e

```
